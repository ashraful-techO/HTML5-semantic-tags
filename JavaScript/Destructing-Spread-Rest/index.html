Destructuring & Spread/Rest

Open console to check the exapmles

  1. Destruct: Extract values
  2. Spread: Copy or expand values
  3. Rest: Collect remaining values
      
Destruct

     
    
       
  1. Array Destruct: const numbers = [1,2,3,4];
        const [a,b,c,d] = numbers;
        console.log(a,b,c,d);
    2. 
        
  2. Obejct Distruct: const user = { name: "Shanto", age: 27 };
        const {name, age} = user;
        console.log(name,age); //Shanto 27
      
    3. 
        
  3. Function Distruct: function greet({name}){
        console.log(`Hello ${name}`);
        }
        greet({name: "Shanto"}) //Hello Shanto;
      
    4. 
     

      
Spread

     
    
      
  1. Array Spread: const arr1 = [1,2];
        const arr2 = [...arr1,3,4];
        console.log(arr2); //1,2,3,4
    2. 
      
  2. Object Spread: const obj1 = { name1: "shanto"};
      const obj2 = {...obj1, name2: "nasif"};
      console.log(obj2); //{name1: 'shanto', name2: 'nasif'}
    3. 
      
  3. Function Spread:  const nums = [1,25,4];
console.log(Math.max(...nums)); //25
    4. 
     

     
     
Rest

     
    
      
  1. Rest in Function:  function sum(...numbers){
  return numbers.reduce((acc,n)=> acc+n,0);
}
      console.log(sum(1,2,3));//6
    2. 
      
  2. Rest in Array const [first, ...rest] = [10, 20, 30, 40];
      console.log(first); // 10
      console.log(rest);  // [20, 30, 40]
    3. 

  3. Rest in Obeject 
const { name, ...rest } = { name: 'Sam', age: 30, city: 'NY' };
console.log(name); // Sam
console.log(rest); // { age: 30, city: 'NY' }
